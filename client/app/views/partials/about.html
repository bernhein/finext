About

JSON, glTF and OBJ file viewer.

Created by Josh Staples ( blog | Git | LinkedIn ) to demonstrate a couple of AngularJS/three.js patterns

There are many ways to work with Angular. That being said, I've kept this project intentionally simple for those of us who are still learning.

First Approach

  • Uses most common Angular coding patterns
  • Minify with Grunt. Use ng-annotate.
  • Uses ['$scope', function($scope){...

Second Approach

  • Uses prototypal pattern for, well, everything.
  • Uses Angular services instead of factories.
  • Uses closure compiler to link and compile code.
